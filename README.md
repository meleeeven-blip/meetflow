# MeetFlow · 视频会议助手

> 覆盖会前准备、AI纪要生成、待办跟进全链路的轻量级会议效率工具

---

## 产品背景

办公场景下，用户普遍面临三个痛点：

- **会议信息分散**：议程、材料、参会人分散在多个工具，准备耗时
- **纪要整理耗时**：会后手工整理录音和笔记，平均需要 60–90 分钟
- **待办事项遗漏**：口头分配的行动项遗漏率高达 40%，缺乏系统跟进

MeetFlow 用 AI 驱动的方式解决以上问题，将会议准备从 1 小时压缩到 5 分钟，纪要生成从 90 分钟压缩到 1 分钟。

---

## 功能模块

### 1. 仪表盘
- 今日会议议程总览，含状态标注（已结束 / 已就绪 / 即将开始）
- 关键数据统计：今日会议数、待办数、本周时长、AI 纪要生成量
- 紧急待办快捷入口，AI 快捷功能面板

### 2. 会前准备
- **月历选会议**：完整月历视图，有会议的日期带蓝点标注，点击日期切换当日议程列表，选中会议自动同步时间段和目标
- **时间段选择**：开始/结束下拉选择器，实时计算并显示会议时长
- **AI 生成议程**：输入会议目标一键生成结构化议程
- **参会人管理**：显示确认状态（已确认 / 待确认）
- **预阅材料**：文件上传与管理，会前提醒阅读
- **准备完成度进度条**：实时显示各项准备状态

### 3. AI 纪要生成
- **左栏**：结构化纪要内容，分三个 Tab
  - 「纪要」：核心决策卡片、风险识别、行动项清单、会议概述
  - 「全文」：完整纪要正文
  - 「Chat」：基于会议内容的 AI 问答
- **右栏**：分三个 Tab
  - 「上传录音」：拖拽或点击上传音频/视频（MP3、M4A、MP4、WAV、MOV），上传后模拟 AI 转录进度（语音识别 → 发言人区分 → 文本生成），自动区分多人发言
  - 「转录文本」：原始逐字稿，按发言人 + 时间戳展示，关键词高亮
  - 「文件回放」：以时间轴形式展示会议中共享的截图/文档，标注共享人和时间节点，点击可跳转至对应转录位置；支持上传新文件追加到时间轴
- **底部音频播放器**：始终可见，支持进度拖拽、倍速切换（0.75x–2.0x）
- **导出格式**：Word (.docx) / PDF (.pdf) / Markdown (.md)，可选择导出内容范围

### 4. 待办跟进
- 行动项按优先级分层展示（紧急 / 本周 / 下周 / 已完成）
- 每项关联来源会议、责任人、截止日期
- 完成进度统计面板
- AI 智能提醒：截止预警 + 跨会议协调建议

### 5. AI 助手
- 基于会议内容的对话式问答
- 快捷指令：起草邀请邮件、总结核心结论、列出风险、催办提醒

### 6. 历史归档
- 所有历史会议列表，支持按时间、纪要、待办筛选
- 一键跳转查看对应纪要

## 特色功能

### 🌙 深色 / 浅色模式
侧边栏底部点击月亮图标切换，偏好自动保存，刷新后保持。

### 📱 发送到手机
纪要页和待办页均支持，点击「发送到手机」弹出操作面板：

| 选项 | 说明 |
|------|------|
| 📅 导出到日历 | 生成 `.ics` 文件，手机点击自动导入系统日历并设置提醒 |
| 📋 复制到备忘录 | 格式化内容一键复制，粘贴进手机备忘录 App |
| 🔗 分享链接 | 复制页面 URL，发给自己手机直接打开 |

> iOS：下载 .ics 后点击「用日历打开」；Android：点击后选择系统日历 App

### 🎙️ 录音上传与 AI 转录
- 支持拖拽或点击上传音频/视频文件
- 上传后模拟 AI 转录流程，分阶段展示进度（语音识别 → 发言人区分 → 生成文本）
- 转录完成后自动更新纪要内容，准确率 96.8%

### 📎 会议文件回放
- 时间轴形式展示会议中共享的截图和文档
- 每条记录标注时间戳、共享人、文件类型
- 点击文件跳转至转录文本中对应的时间节点
- 支持上传新文件，实时追加到时间轴

## 设计思路

围绕「会前 → 会中 → 会后」三阶段用户旅程设计：

```
创建会议 → AI生成议程 → 发送邀请 → 上传录音 → AI生成纪要 → 分配待办 → AI追踪提醒
```
